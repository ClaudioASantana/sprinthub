# Gerenciar Empresas (Tenants)

## Descrição
Como Administrador Global autenticado,
Quero listar, visualizar detalhes e adicionar novas emrpesas (Tenants) clientes cadastradas no SprintHub,
Para conseguir gerenciar contratantes do SaaS sem a necessidade de intervenção direta no Banco de Dados.

## Critérios de Aceite
- [x] A tela principal (Dashboard) do painel deve exibir um atalho para a listagem principal de "Empresas Clientes".
- [x] A listagem deve retornar um array de empresas cadastradas no backend principal do SprintHub.
- [x] O Administrador deve possui um botão para cadastrar/editar os dados básicos da empresa (Nome da Empresa, CNPJ/ID, Contato Responsável).
- [x] O Administrador deve conseguir ativar ou inativar uma empresa bloqueando assim, todos os sub-usuários vinculados a ela.
